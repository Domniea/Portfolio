A Few Projects

Here are some examples demonstrating my work.

The Best/Forget Forgetting ToDo list - Mobile Application

An Ios/Android app built with React Native. Utilizing AWS's Amplify and Cognito services, the app has a full authentication flow configured with a custom UI and components. "The Best/Forget Forgetting ToDo List" is a theme responsive user friendly utility tool to keep a list of notes/descriptions. Currently published on the Apple App Store and Google Play Store.

Job Search Website

Here is an example of a responsive job search application. With a React driven Frontend and Express driven backend, this application utilizes JSONWeb tokens to impliment strong authentication. Users can create their own profile and post available jobs along with a price point. Potential workers will be able to view posted jobs and choose whether to accept. Once a posted job has been accepted, the job poster will receive a notification and the option to sign-off upon completion of the job. Signing off on the job will remove it from the public queue and add the payment amount to the worker's profile. As "SuperUser" or Management, the user is able to search for users via their email to convert their profile into a potential worker which will enable them to accept posted jobs.

Marvel SuperHero and Movie(MCU) Website

A representation of React Front-End development. This application demonstrates use of state, API data retrieval, and data manipulation. Based on the day of the year, two heros are selected and displayed. The user can then see who would win in a fight(based on the characters stats). The user may then browse all Marvel Movies to find a small description, movie poster image, and a link to the movie trailer. Lastley, the user can browse over 270 superheros grouped by the first letter of their name and see that heros stats.

"Dog Walker Company App"

A mock React application of a dog walking company. The Front-End utilizes React, while the backend is a mixture of Express/Mongoose/mongoDB. Database information is stored using MongoDB Atlas. List of user added Dogs are also searchable by day of the week(to be walked) using Querey Strings. Please feel free to add a dog to the list or youreself/someone to the Staff List. User Authentication/Authorization coming soon.

Discussion Forum

Based on the concept of many forum/chat applications. "Rock The Vote" allows users to create a profile and discuss or debate any given subject. Users are not only allowed to post on the form, but they are able to comment on others posts and "Up Vote" and "Down Vote". This application utilizes React on the Frontend and Express/MongoDB Atlas as the backend.
